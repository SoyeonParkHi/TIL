* Up & Down 숫자맞추기 게임  
1~100 사이의 정답 숫자를 랜덤으로 하나 정하고, 내가 입력한 숫자가 정답보다 작으면 'DOWN', 정답보다 크면 'UP'을 출력하고 숫자를 다시 입력받고, 정답을 맞추면 '정답!'이라고 출력하고 게임을 끝낸다. 기회는 5번이고 5회가 넘으면 '횟수초과' 메시지와 함께 정답을 알려줍니다. 정답을 맞추면 몇번째에 맞추었는지 출력합니다.
```python
import random # random 모듈(파이썬 기본함수 아님)
n = random.randrange(1,101) # 범위 내의 랜덤 숫자 생성(=정답)
for i in range(1,6) :
    a = int(input('숫자를 입력하세요:'))
    if a == n :
        print(f'정답!{i}번째에 정답을 맞추었습니다!')
        break
    elif a >= n :
        print('DOWN')
    else : 
        print('UP')
if i >= 5 :
    print(f'횟수초과! 정답은 {n}입니다')
```

```python
score_list = [[96,84,80],[96,86,76],[76,95,83],[89,96,69],[90,76,91]]
```

```python
영어{score_e}
수학{score_m}''')

avg_k = sum(score_k)/len(score_k)
avg_e = sum(score_e)/len(score_e)
avg_m = sum(score_m)/len(score_m)
print(f'''국어 평균 : {round(avg_k,1)}
영어 평균 : {round(avg_e,1)}
수학 평균 : {round(avg_m,1)}''')
```

## 키와 값의 쌍으로 된 데이터, 딕셔너리
### 딕셔너리 만들기
* dict 함수로 딕셔너리 만들기
    - 딕셔너리명 = dict(키1=값1, 키2=값2,...)
        + **키에 따옴표를 쓰지 x** (딕셔너리가 생성되면서 자동으로 문자열형으로 저장된다)
    - 딕셔너리명 = dict(zip(key리스트, value리스트))
    - 딕셔너리명 = dict([(키1,값1),(키2,값2),...])
    - 딕셔너리명 = dict({키1:값1,키2:값2,...})

* 딕셔너리 자료형
    - value에는 모든 자료형 혼합 사용 가능
    - key에는 숫자, 문자열, 부울형, 튜플 사용 가능

### 딕셔너리 값 추출
* 인덱스가 아닌 키(key)로 자료에 접근
    - 딕셔너리명[key] : 존재하지 않는 키로 접근하면 오류
    - 딕셔너리명.get(key,msg) : 존재하지 않는 키로 접근하는 경우 msg 반환

### 딕셔너리 항목 추가/수정
* 딕셔너리명[key] = value : 키가 존재하지 않으면 추가, 존재하면 수정
* 딕셔너리명.setdefault(key,value) : 키가 존재하지 않으면 추가(키가 존재해도 수정되지 x)
* 여러 항목 추가/수정하기
    - dict 함수와 사용법 동일하게 update 함수 사용
    - 키가 존재하면 수정, 존재하지 않으면 추가된다.
    - 딕셔너리명.update(키1=값1, 키2=값2,...)
    - 딕셔너리명.update(zip(key리스트, value리스트))
    - 딕셔너리명.update([(키1,값1),(키2,값2),...])
    - 딕셔너리명.update({키1:값1,키2:값2,...})

### 딕셔너리 항목 삭제
* del 딕셔너리명[key] : 해당 키의 항목 삭제
* 딕셔너리명.clear() : 딕셔너리의 모든 항목 삭제
* 딕셔너리명.pop(key) : 해당 키의 값 리턴 후 삭제
    - 키가 존재하지 않으면 오류
    - 딕셔너리명.pop(key,msg) : 키가 존재하지 않을 때 msg 반환

### 딕셔너리의 키, 값을 리스트 형태로 가져오기
* 딕셔너리.keys() : 딕셔너리의 key만 리스트 형태로 받아서 사용
* 딕셔너리.values() : 딕셔너리의 value만 리스트 형태로 받아서 사용
* 딕셔너리.items() : 딕셔너리의 (key,value)를 리스트 형태로 받아서 사용

### for문에서 딕셔너리의 key, value 사용하기
* for i in 딕셔너리명.keys() / for i in 딕셔너리명 : print(i)하면 딕셔너리의 키 출력
* for i in 딕셔너리명.values()
* for i in 딕셔너리명.items() : print(i)를 하면 딕셔너리의 (key,value) 출력
* for k,v in 딕셔너리명.items() : print(k,v)를 하면 딕셔너리의 key는 k에, value는 v에 담아 출력

### 딕셔너리 정렬
* 리스트 정렬 함수인 sorted 사용
* sorted(딕셔너리명.keys())
* sorted(딕셔너리명.values())
* sorted(딕셔너리명.items()) : 딕셔너리의 (key,value)를 리스트 형태로 받아서 key 기준으로 정렬
* .sort()는 지원하지 않음

### 딕셔너리 연습문제
* 영어 단어장 만들기  
엔터를 입력할 때까지 영어단어, 뜻을 입력받아 단어장을 만들고, 입력이 끝나면 단어 테스트를 실시하는 프로그램을 만들어봅시다.  
1) 단어장을 만듭니다.
2) 단어테스트를 실시하고 맞은 갯수를 계산합니다.
3) 테스트가 끝나면 맞은갯수/전체단어수/점수 형태로 결과를 출력합니다.
```python
words = {}
while 1 :
    input_w = input('영어단어,뜻 : ')
    if input_w == '' :
        break
    a,b = input_w.split(',')
    words[a] = b
cor = 0
for i,j in words.items() :
    answer = input(i)
    if answer == j :
        cor += 1
print(f'맞은 갯수는 {cor}개, 전체문제수는 {len(words)}개, 점수는 {round(cor/len(words)*100,1)}점입니다.')
```
