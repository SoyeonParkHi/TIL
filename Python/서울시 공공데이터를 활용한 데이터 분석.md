### 알고리즘 연습하기
* Up & Down 숫자맞추기 게임  
1~100 사이의 정답 숫자를 랜덤으로 하나 정하고, 내가 입력한 숫자가 정답보다 작으면 'DOWN', 정답보다 크면 'UP'을 출력하고 숫자를 다시 입력받고, 정답을 맞추면 '정답!'이라고 출력하고 게임을 끝낸다. 기회는 5번이고 5회가 넘으면 '횟수초과' 메시지와 함께 정답을 알려줍니다. 정답을 맞추면 몇번째에 맞추었는지 출력합니다.
```python
import random # random 모듈(파이썬 기본함수 아님)
n = random.randrange(1,101) # 범위 내의 랜덤 숫자 생성(=정답)
for i in range(1,6) :
    a = int(input('숫자를 입력하세요:'))
    if a == n :
        print(f'정답!{i}번째에 정답을 맞추었습니다!')
        break
    elif a >= n :
        print('DOWN')
    else : 
        print('UP')
if i >= 5 :
    print(f'횟수초과! 정답은 {n}입니다')
```

* 사칙연산 프로그램  
두 수와 사칙연산기호(+,-,*,/)를 입력받아 연산 기호에 따라 연산 결과를 출력하는 프로그램. 사칙연산기호(+,-,*,/)가 아닌 경우 '잘못 입력하셨습니다' 출력
```python
a = int(input('숫자 1 : '))
b = int(input('숫자 2 : '))
c = input('사칙연산기호(+,-,*,/) : ')
if c == '+' :
    print(f'{a}{c}{b}={a+b}')
elif c == '-' :
    print(f'{a}{c}{b}={a-b}')
elif c == '*' : 
    print(f'{a}{c}{b}={a*b}')
elif c == '/' :
    print(f'{a}{c}{b}={a/b}')
else : 
    print('잘못 입력하셨습니다')
```

* 할인된 금액 계산  
물건 구매가를 입력받고, 금액에 따른 할인율(10만원 이상 : 10%, 5만원 이상 : 7%, 1만원 이상 : 5%)을 계산하여 구매가, 할인율, 할인금액, 지불금액을 출력
```python
price = int(input('구매가를 입력하세요 : '))
print(f'구매가 : {price}')
if price >= 100000 :
    discount = 10
elif price >= 50000 :
    discount = 7
elif price >= 10000 :
    discount = 5
else : 
    discount = 0
print(f'''할인율 : {discount}%
할인금액 : {price*discount/100}
지불금액 : {price*(100-discount)/100}''')
```

## 데이터 처리의 기초 리스트와 튜플

### 파이썬의 복합자료형
* 하나의 변수에 다수의 데이터 저장
* 리스트(list)  
x=[1,2,3,4,5]  
순서 有, 인덱스로 제어, 중복 허용
* 튜플(tuple)  
x=(1,2,3,4,5)  
리스트와 유사, 변경 불가능, 괄호 생략 가능, 항목 1개일 경우 마지막에 , 추가
* 딕셔너리(dict)  
x={'이름':'홍길동', '나이':20}  
순서 無, 키:값 쌍으로 이루어짐, 키로 제어
* 집합(set)  
x={1,2,3,4,5}  
순서 無, 중복 배제

### 리스트와 튜플
* 모든 자료형 혼합 저장 가능
* \+ : 리스트, 튜플 연결
* \* : 리스트, 튜플 반복
* 멤버연산자(in, not in) : 리스트, 튜플에 값 존재여부 확인
* len : 리스트, 튜플 항목(item)의 갯수 구하기

### 중첩리스트/튜플 인덱싱  
ex)
```python
# 아래 리스트 l에서 '수박' 인덱싱하여 출력하기
l = ['사과', '오렌지', '포도', ['수박', '바나나']]
print(l[3]) # ['수박','바나나']가 출력
print(l[3][0])
```

### 슬라이싱  
* 리스트/튜플명[시작인덱스:끝인덱스:간격]
* 끝인덱스 앞에서 슬라이싱
* 역순으로 출력할때는 간격을 -1로 하고 인덱싱도 끝에서부터 계산
ex)
```python
l=['축구','농구','배구','야구','족구','발야구','피구']
print(l[-2:-5:-1]) # '발야구(-2)','족구(-3)','야구(-4)' 출력
```
* 항목의 인덱스 찾기  
리스트명/튜플명.index(항목) : 특정 항목의 인덱스를 추출

### 리스트 변경
* 리스트에 항목 추가하기
    - 리스트명.append(항목) : 리스트의 맨 마지막에 항목 추가
    - 리스트명.insert(인덱스, 항목) : 지정한 인덱스에 항목을 삽입
    - 리스트명.extend(리스트) : 리스트 마지막에 여러 항목 한번에 추가

* 리스트의 항목 수정하기
    - 리스트명[index] = item : 지정한 인덱스의 항목 변경
    - 리스트명[시작인덱스:끝인덱스]=[item1,item2,...] : 여러 개 항목 한번에 변경

* 리스트의 항목 삭제하기
    - del 리스트명[인덱스] : 지정한 인덱스의 항목 삭제
    - 리스트명.remove(항목) : 항목값을 삭제
    - 리스트명.pop() : 리스트 마지막 항목을 반환하고 삭제

### 리스트 활용하기
* sum, min, max(리스트명)
* 리스트명.count(항목) : 항목의 갯수
* 리스트 정렬
    - 오름차순 정렬(원본 정렬) : 리스트명.sort()
    - 내림차순 정렬(원본 정렬) : 리스트명.sort(reverse=True)
    - sorted(리스트명) : 원본 수정 x
    - sorted(리스트명, reverse=True) : 원본 수정 x, 역순
    - 리스트명.reverse() : 리스트 역순, 원본 수정 x
